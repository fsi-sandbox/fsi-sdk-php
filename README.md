# Sandbox PHP SDK

Welcome to the FSI Sandbox SDK for PHP. See the [fsi-sdk topic](https://github.com/topics/fsi-sdk) for other available SDKs

### Installation

```bash
$ composer require innovation-sandbox/innovation-sandbox
```
